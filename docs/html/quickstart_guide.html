2. Quick Start Guide

2.1. Retrieve Analyzed Data

At first, load nifs.retrieve.anadata.AnaData class:

[1]:

from nifs.retrieve import AnaData

And initialize AnaData class with parameters: shotnumber : 80000, subshot number : 1

[2]:

ana = AnaData(80000, 1)

If you would like to search for registerd diagnostics in the kaiseki-server, the property diagnostics_list is useful because it returns a dict of key: diagnostics name and value: description:

dataset = ana.retrieve("ha1")
